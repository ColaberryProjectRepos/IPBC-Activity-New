Adding KPIs 

1.  Drag and drop a Card visual from the visuals in the canvas.
2.  Drag and drop Student column in the Fields under Visualization and SUM.
3.  Under Visual tab, change Callout value Font to DIN, Size 25, Color Black.
4.  Under Properties set size to Height 74, Width 184, and set Position to Horizontal 2, Vertical 113.
5.  In General tab, change Title to “Student Days”, Heading 3, Font to DIN, Font size to 13, Color White, align Center, and Background color to #8495A8.
6.  Duplicate the 1st card 3 more times to create the remaining KPIs cards
7.  Name as follow, 2nd card “Wks in Prog”, 3rd card “Prev Day % Inc”, 4th card "% Sec on Pace.”
8.  Change Fields as follows: all fields are AVERAGED for 
a.  2nd card: WksInProgramInt 
b.  3rd card: PrevDayPerInc   
c.  4th card: % Section
Creating indicator arrows for 3rd and 4th cards
1.  Add another card visual to the canvas and add Arrow Wks In Progtb2 to the field.
2.  In formatting in the Visual tab change front to Size 20, and under General tab change Properties size to Height 31 and Width 40. Place the next to 3rd KPI to right of the number.
3.  Repeat step 3 for 4th KPI arrow and change field to Arrow Section on Pace
