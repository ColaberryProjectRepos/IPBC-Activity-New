Creating Other Matrix tooltips


1.       Click to add a new page, name it Tool_Tips3
2.       In the Formatting Tab-->Page information--> Name: Tool_Tips2, Allow use as tooltip: On, Canvas settings: Custom, Height: 250pix, Width: 812pix Vertical Alignment: Top.
3.       Drag a Table inside the tooltip box on the canvas.
4.       Add the following columns and measures one by one to the Columns field
a)      MM_Student – from ADF_Mentorship_Activity
b)      LastHWsubmittedDays – from ADF_Mentorship_Activity rename it as Days On Curr HW then right click and select Average
c)       PerComp_Act - from All Measures rename it as % Complete
d)      Avg PerOnPace - from All Measures rename it as % On Pace then right click and select conditional formatting then click on Font color -->Format style =Rules, What field should we base this on? = Avg PerOnPace
If value >=0 and <0.9 then = Red #ED092B
If value >=0.9 and <1 then = Yellow #F0EC35
If value >=1 and <100 then = Green #52C62F
e)      WksInProgramInt – from ADF_Mentorship_Activity, rename it as Wks In Program
f)       CurrSection_Phase1 - from ADF_Mentorship_Activity
g)      MM_Mentor - from ADF_Mentorship_Activity
h)      MentorScore - from ADF_Mentorship_Activity, then right click and select conditional formatting then click on font color -->Format style =Rules, What field should we base this on? = Sum of Mentors
If value >=0 and <0.75 then = Red #E41818
If value >=0.75 and <=Max then = Green #13DE16
5.       In the formatting Tab -->Visual-->Grid; Horizontal gridlines: On, Color: #E5E5E5, Width: 2, Vertical gridlines: On, Color: #E5E5E5, Width: 1, Values: Font: Segoe UI: 10, Color: #252423.
6.       In the Formatting Tab-->General-->Properties-->Size: Height = 203, Width = 812, Position: Horizontal = 0, Vertical = 0.
