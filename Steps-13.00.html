Overview - Adding Scatter Plot Chart 13


1.       From the visualization pane drag a Scatter chart into the canvas.
2.       Fill the fields as follows:
a.       Values - CurrSection_Phase1
b.       X axis  - SectionOrder
c.       Y axis - Avg PeronPace
d.       Size - Students
e.       Tooltip - LastRunDate, Avg Wks In tb2,PrevDayPerInc

3.       In formatting tab under Visual change X axis Values for Font to Size 9, Color #605E5C, and under Title change Title text to “Section”, Font to Size 12, Color Black, 
4.       Repeat the same steps for Y axis but change Title text to “Auto.”
5.       Gridlines: for both Horizontal and Vertical change Style to Dotted, Color #C8C6C4, Width 1px.
6.       Markers: change Shape Type to Circle, Size -10px, Color to conditional formatting as so



7.       Turn Category label ON and set Font to 9, Color to #808080. 
8.       In General tab set Height to 345, Width to 1094, and position according to final report pic at the top of the page. 
9.       Title: text “Section | on Pace”, Heading 3, Font size 14, Color White, Background color #8495A8, align Center, 
10.     Tooltip: set Type to Report page, page to Auto, Text size 10, all Color to Black, Background White, Transparency 0
