Overview - Adding Matrix 14


1.       Add a Matrix to the canvas and fill the fields as follows:
a.       Rows - DaysBack Group, DaysBack, Wks In Group
b.       Columns- SectionOrder
c.       Values - AVG Wks In tb2
2.       From formatting tab, under Visual change: 
3.       Values: Font Segoe UI, Font size 12, Color black, background color White.
4.       Column Headers: change Font to Segoe UI, Size 10, Color White, background color #8495A8.
5.       Row Headers: Font Segoe UI, size 10, color Black, background color White, align Center.
6.     Cell Elements: series “AVG Wks In tb2”, background color conditional formatting 



7.       Formatting General tab:
8.       Properties: set size to Height 262, Width 1278, position Horizontal 2, Vertical 458.
9.       Tooltip: type Report Page, change page to Tool_Tip2, change text to font Segoe UI, size 10, color Black.
